Chip variant hypothesis · in context
Blue · forward action · "Send a reminder" / "Reschedule"
External completed actions where reversal is not available. The chip offers a context-aware next step — not an undo. Blue is consistent with staged/pending semantics: there's more to do, and the system has a suggestion. Label is a plain verb phrase, not "undo."
External · irreversible · next step offer
Amber · reversal · "Remove buffer"
Internal completed action — calendar buffer is fully reversible. Amber maps to held/caution semantics: the system did something that can be undone if the user disagrees. Label pattern is "Remove / Undo [action]" — unambiguous about what happens on tap. This is the first amber chip in the design system; legibility against the warm dark background holds.
Internal · reversible · genuine undo · amber first use
Green · system offer · "Preview draft"
Staged item awaiting user action — nothing sent, nothing irreversible yet. Green signals the system's recommendation. Consistent with "acted/done" semantics: Aether has done its part; this is the handoff moment. "Preview draft" names what the user sees on tap, not what the system did.
Staged · system recommendation · handoff
Pre-attentive scan reads correctly
Scanning the four entries: green → green → amber → blue. The amber entry is visually distinct before reading. A user who has learned the colour semantics can triage at a glance — "that's something I can undo" — without processing the label. Consistent with Ford Williams' pre-attentive encoding principle.
